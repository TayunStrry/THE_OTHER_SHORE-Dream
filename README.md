## 安装教程

- **购买并安装 minecraft Bedrock 的最新< 正式版 >与< 测试版 >**
- **下载并放置本git仓库的代码数据**
- **在 Visual Studio Code 中下载并安装 npm | NodeJS | TSC**
- **尝试进行 TS 脚本编译** : 在终端中输入：
```bash
## 使用预览版时，使用以下命令进行编译：
npm run compile

## 使用正式版时，使用以下命令进行编译：
npm run compile_stability
```
